For advanced control of playback and other interface elements, Banshee supports key bindings, or combinations of keys (and/or clicks), that can be pressed to perform an action.





### Playback Control








  Key
  Action





  
Enter, o

  
If one or more items are selected, play the first item in the selection





  
Space Bar

  
Play or Pause the current song





  
N

  
Play the next song





  
B

  
Play the previous song






### Library Interaction








  Key
  Action



  
/, Ctrl + F

  
Move focus to the search box





  
Control + I

  
Open import media dialog






### Playlists








  Key
  Action



  
Control + N

  
Create new playlist






### Podcasts








  Key
  Action



  
y

  
Mark the selected episodes as old






### Interface








  Key
  Action




  
F

  
Toggle full-screen mode






  
Control + A

  
Select all songs in playlist view





  
Control + Shift + A

  
Unselect all songs in playlist view





  
Control + W

  
Hide Banshee Window (Requires Notification Area Plugin Enabled)





  
Control + Left Click

  
Play Previous Song  (Requires Notification Area Plugin Enabled)






  
Control + Right Click

  
Play Next Song (Requires Notification Area Plugin Enabled)





  
Control + Middle Click

  
Toggle Play/Pause (Requires Notification Area Plugin Enabled)
